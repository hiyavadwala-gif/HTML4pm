  • Scope of Variables
  • *Types of Scope Scope Type Description
  • Global Scope- Variables declared outside any block/function. Accessible everywhere.
  • Local Scope- Variables declared inside a function. Accessible only within that function.
  • Block Scope- Variables declared inside { } using let or const. Accessible only within that block.
