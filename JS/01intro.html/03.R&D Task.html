  • Q6. Research difference between HTML, CSS, JS.
  • HTML provides the foundation and structure for every web page. It defines the content and organization using tags.
  • CSS is the styling language used to control the presentation, formatting, and layout of the content defined by HTML. It makes the website look good.
  • JavaScript is the programming language that enables interactive web pages. It is what allows the page to do things after it has loaded.

  • Q9. Search how JavaScript changed the web.
  • JavaScript has fundamentally transformed the web from a collection of static, document-like pages into the dynamic, interactive application platform we use today. Before JavaScript, websites were slow, and users had to reload the entire page every time they clicked a button or submitted a form.

  • Q10. Explore what Node.js is and where it runs.
  • Node.js is an open-source, cross-platform JavaScript runtime environment that allows developers to execute JavaScript code outside of a web browser
